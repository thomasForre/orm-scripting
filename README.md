# orm group script
Script for automating grouping of columns in work permit search page in ORM

## Grouping
The code is a script with a small UI that makes it possible to save grouping of columns in work permit search page in ORM, and let you re-group to your selected favorite in one single click.
#### Setup
1. Copy the raw content of the `orm-grouping-script.min.js` file 📄
2. In you browser, add a new bookmark (favorite) ⭐
3. In the URL field of that new bookmark, paste the entire content of the file – name it whatever you like 🐍

#### Use
Navigate to ORM work permit search page, and click the newly added bookmark – a popup card will appear.
<br>
Select and move checkboxes up and down according to your preferences and click → `✅ Bruk`.
<br>
The columns will group accordingly and the setup is stored.
<br>
Next time you click the bookmark, the colmns will group this way.

To change settings, click the gear icon above the columns headers.
