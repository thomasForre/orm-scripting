# orm group script
Script for automating grouping of columns in work permit search page in ORM

## Grouping
The code is a script with a small UI that makes it possible to save grouping of columns in work permit search page in ORM, and let you re-group to your favorite setup in one single click.
#### Setup
1. Copy the raw content of the `orm-grouping-script.min.js` file 📄
2. In your browser, add a new bookmark (favorite) ⭐
3. In the URL field of that new bookmark, paste the entire content of the file – name it whatever you like 🐍

#### Use
1. Navigate to ORM work permit search page, and click the newly added bookmark – a popup card will appear:
<img width="279" height="423" alt="image" src="https://github.com/user-attachments/assets/6ae2f5bc-c4a5-40f1-a658-ad4765070537" />

2. Select and move checkboxes up and down according to your preferred order and click `Bruk`. The popup card will close, and the columns will group accordingly.

3. If you want to save the grouping as default, you need to click the save button `💾`. Next time you click the bookmark, the columns will group according to the saved setup.

To change settings, click the gear icon above the columns headers.
